# Ring Protocol v4

- [Overview](https://docs-ring-protocol.vercel.app/contracts/v4/overview): Ring v4 combines concentrated liquidity with hooks, singleton architecture, and gas optimizations across the pool lifecycle.
- [Hooks](https://docs-ring-protocol.vercel.app/contracts/v4/concepts/hooks): Ring v4 hooks allow developers to attach custom logic to pool initialization, liquidity, swaps, and donations.
- [Deployments](https://docs-ring-protocol.vercel.app/contracts/v4/deployments): Published Ring v4 deployment addresses across supported networks.
- [Few Hook Contracts](https://docs-ring-protocol.vercel.app/contracts/v4/guides/hooks/few-hook-contracts): Few Protocol hook contracts and published Ethereum mainnet hook addresses.
- [Hook Deployment](https://docs-ring-protocol.vercel.app/contracts/v4/guides/hooks/hook-deployment): Hook permission flags, address encoding, and CREATE2 mining workflow.
- [Building Your First Hook](https://docs-ring-protocol.vercel.app/contracts/v4/guides/hooks/your-first-hook): Introductory guide to building a custom hook.
- [Create Pool](https://docs-ring-protocol.vercel.app/contracts/v4/quickstart/create-pool): Create and initialize a Ring v4 pool.
- [Swap](https://docs-ring-protocol.vercel.app/contracts/v4/quickstart/swap): Execute swaps through Ring v4 routing flows.
- [StateView](https://docs-ring-protocol.vercel.app/contracts/v4/guides/state-view): Read pool state using StateView for offchain and onchain integrations.
- [Unlock Callback](https://docs-ring-protocol.vercel.app/contracts/v4/guides/unlock-callback): Understand unlock-based execution and settlement deltas.
- [Custom Accounting](https://docs-ring-protocol.vercel.app/contracts/v4/guides/custom-accounting): Extend pool behavior with custom accounting logic.
- [Swap Routing](https://docs-ring-protocol.vercel.app/contracts/v4/guides/swap-routing): Route swaps across Ring v4 pools and related routers.
