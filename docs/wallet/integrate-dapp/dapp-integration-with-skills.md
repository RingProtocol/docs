This guide shows the fastest way to integrate your DApp with Ring Wallet using Skills.

## 1. Add Ring AI Skills

Run:

```bash
npx skills add ringprotocol/ring-ai
```

## 2. Select and Install the Skill

Choose the skill:

```text
dapp-to-ringwallet
```

Then complete the install flow in the prompt.

![Add skill and select dapp-to-ringwallet](/img/skill/add_skill_and_select%20dapp_to_ringwallet.png)

## 3. Integrate in Your IDE or CLI

Open your IDE or CLI and follow the installed skill steps to wire Ring Wallet integration into your DApp.

You can use the skill-generated integration actions to:

- connect wallet flows
- provider setup
- request handling

After finishing, run your app locally and verify the Ring Wallet connection flow works end-to-end.
