## The Ring Core SDK

The Ring Core SDK provides abstractions for other SDKs to use in a Typescript/Javascript environment.

This SDK is not expected to be used in isolation, but only as part of other SDKs.

We recommend taking a look at the [**Technical Reference**](./reference/overview).

Some autogenerated API reference pages in this section still retain upstream source-link formatting from the original generator output. For Ring integrations, the package you should install is `@ring-protocol/sdk-core`.
