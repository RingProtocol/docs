title: Ring Swap v2 Deployments
---

Ring Swap v2 deployment addresses are maintained on the canonical
[Ring Protocol Contracts](/contracts/v2/deployments) page.

Use that page for the current Ring Swap factories, Ring Swap routers, Few factories, UniversalRouter,
Permit2, wrapper contracts, hook contracts, and init code hashes by network.

Do not use upstream v2 deployment address tables for Ring Swap v2 integrations. Ring Swap v2
uses Ring-managed deployments built around FewToken wrapping.
