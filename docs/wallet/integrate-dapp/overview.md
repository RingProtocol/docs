# Integrate DApp

This section is the entry point for integrating your DApp with Ring Wallet.

## Integration Flow

* [Integrate DApp manually](./dapp-integration.md). *OR* [Integrate with skill](./dapp-integration-with-skills.md)
* [Register app](./register-your-dapp.md)
 - DApp logo url
 - DApp name
 - DApp Url
* [Test app](./test-your-dapp.md)

## What You Will Complete

- Register your DApp and get an API key
- Integrate the Ring Wallet SDK or use Skills
- Verify wallet connect, transactions, and signatures
