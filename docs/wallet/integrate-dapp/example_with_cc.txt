### 3. Wagmi 集成
- useAccount hook 从 useAccountWagmi() 获取连接状态
- injectedWithFallback connector 会检测 window.ethereum
- 当 dappsdk.js 加载后， window.ethereum 会被设置为 RingWalletProvider
- wagmi 会自动通过 EIP-6963 发现 Ring Wallet
## 工作原理
当 dapp 运行在 Ring Wallet 的 iframe 中时：

1. dappsdk.js 注入 window.ethereum (RingWalletProvider)
2. 通过 postMessage 与父窗口 (Ring Wallet) 通信
3. Ring Wallet 已连接时， useAccountWagmi() 会返回 isConnected: true
4. account.isConnected 和 account.address 就会有值

### 4. proxy mode
x-forwarded-proto and x-forwarded-host are headers typically set by reverse proxies (Nginx, Vercel, Cloudflare, etc.) when forwarding requests to upstream servers.
In this context:
1. Direct browser request → No reverse proxy → These headers won't exist
   - Falls back to https://localhost which is wrong for production
2. Behind a reverse proxy (e.g., Vercel) → Headers are set by the proxy
   - Works correctly
